Given the following relational schema, write queries in SQL to answer the English question. customer(cid: integer, cname: string, address: string, city: string, state: string) product(pid: integer, pname: string, price: currency, inventory: integer) shipment(sid: integer, cid: integer, shipdate: DateTime) shippedproduct(sid: integer, pid: integer, amount: integer) Return the top 5 products for state 'BC' based on shipment revenue (amount*price) for the year 2021.

+-------+------+-------------------------+-----------+ | state | year | pname | revenue | +-------+------+-------------------------+-----------+ | BC | 2021 | Sports Car | 247000.00 | | BC | 2021 | Table | 6500.00 | | BC | 2021 | Textbook | 500.00 | | BC | 2021 | Deluxe Sweet Collection | 326.50 | | BC | 2021 | Teddy Bear | 103.92 | +-------+------+-------------------------+-----------+ SELECT p.pname AS product_name, SUM(sp.amount * pr.price) AS shipment_revenue FROM customer c JOIN shipment s ON c.cid = s.cid JOIN shippedproduct sp ON s.sid = sp.sid JOIN product p ON sp.pid = p.pid WHERE c.state = 'BC' AND YEAR(s.shipdate) = 2021 GROUP BY p.pname ORDER BY shipment_revenue DESC LIMIT 5;
