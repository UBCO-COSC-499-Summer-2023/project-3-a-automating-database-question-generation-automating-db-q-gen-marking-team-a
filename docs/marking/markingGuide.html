Assignment 2 - Marking Guide

Question 1 (10 marks)

Generally award 1/2 marks wherever possible. Mark breakdown:

  • Airplane table (1 total) - +0.5 for attributes, +0.5 for primary key
  • Airport table (1 total) - +0.5 for attributes, +0.5 for primary key
  • Flight table (4 total) - +1 for attributes, +1 for primary key, +2 for foreign keys
  • Passenger table (1 total) - +0.5 for attributes, +0.5 for primary key
  • OnFlight table (3 total) - +1 for attributes, +1 for primary key, +1 for foreign keys

Note that FK of OnFlight table must have both attributes and must be done in a single statement.
